================================================================================
多策略交易信号系统 - 使用说明
================================================================================

【软件简介】
本软件包含两个独立的交易策略，通过标签页切换使用：

策略1：IF300 股指期货策略 (V10.14)
- 基于IF300股指期货季月合约
- 历史表现（2018-2025）：年均收益135.8%，最大回撤-27.8%

策略2：周末效应策略 (V9)
- 基于创业板ETF的周末效应
- 独立的数据源和交易逻辑

================================================================================
【运行方式】
================================================================================

方式一：直接运行Python脚本（推荐开发调试）
----------------------------------------
1. 确保已安装 Python 3.8+
2. 安装依赖: pip install pandas numpy matplotlib
3. 运行: python signal_app_main.py

方式二：运行打包后的可执行文件
----------------------------------------
Windows用户：
1. 下载 "多策略交易信号系统-Windows"
2. 将 data 文件夹放到 exe 同目录下
3. 双击运行 "多策略交易信号系统.exe"

Mac用户：
1. 下载 "多策略交易信号系统-Mac"
2. 将 data 文件夹放到 app 同目录下
3. 双击运行 "多策略交易信号系统.app"

方式三：本地打包
----------------------------------------
Windows: 双击运行 build_windows.bat
Mac: 运行 ./build_mac.sh

================================================================================
【数据文件说明】
================================================================================

程序需要以下数据文件（位于 data 文件夹）：

IF300策略所需：
- IF_主连_季月合约连接_day.csv  （IF主连日线数据）
- 000300_沪深300_day.csv        （用于预热MA60，可选）

周末效应策略所需：
- 159915_创业板ETF_day.csv      （创业板ETF日线数据）

数据格式要求：
- CSV格式，UTF-8编码
- 必需列：日期,开盘,最高,最低,收盘
- 日期格式：YYYY-MM-DD

================================================================================
【IF300策略规则】
================================================================================

做多策略：
- 开仓日：周三、周四（交割周周四除外）
- MA60比率：99% ~ 110%
- 持仓天数：3个交易日
- 止损：2.0%

做空策略A（非12月）：
- 开仓日：周一
- MA60比率：98% ~ 110%
- 持仓天数：4个交易日
- 止损：1.5%

做空策略B（仅12月）：
- 开仓日：周五
- MA60比率：<= 100%
- 持仓天数：5个交易日
- 止损：2.0%

冷静期规则：
- 单笔亏损超过20%后，7天内不开同方向仓位
